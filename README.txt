=======

** Assets/Scenes/title を実行すれば、ゲームがテストプレイできます **

=======

・fileフォルダには、プログラム担当以外の作業データをそれぞれ置いていきます
・プログラム担当は、対応プラットフォームを随時iOSにして作業を行ってください
・割り当てフラグ(0~127)を使用したい場合は、都度プログラム担当間で話し合ってください

=======

担当内で管理把握するため、プロジェクト部分は以下のように使用しましょう
足りない要素等があれば、随時ディレクトリの追加をお願いします

 Animations	- 各種アニメーション系
 Editor		- エディタ系を司るためのコード
 Fonts		- フォント(現在は適当に拾ってきたもの)
 Materials	- Unity内で作成した物体(現在は空)
 OtherAssets	- バーチャルパッドとフラグ系のソースコード(ごちゃっとしてるのはここ)
 Prefabs	- 複数のアイテムをひとまとめにしたもの(現在は面倒なので空)
 Resources	- 外部参照用(現在テストが置いてあります)
 Scenes		- 現在の作成状態を保存しています。とりあえずこれを実行してみてください。
 Scripts	- 作成したメイン機能のソースコード(随時変更予定)
 Sounds		- ゲーム内SE,BGM
 Sprites	- 各種素材(主に画像)
 Textures	- テクスチャ系(現在は空)


現在、ヒエラルキー部分(Stage)には、以下のものが置いてあります。

 Player		- 主人公に関する設定
 Canvas		- キャンバス。十字キーやテキストウィンドウ。
 Objects	- オブジェクト(家具類)
 ItemList	- オブジェクト(アイテム)
 Wall		- 壁
 MainCamera	- カメラ(視点用)。主人公を追随します。
 AudioSource	- BGM再生
 EventSystem	- 入力や衝突判定の対応
 FlagManager	- フラグの状態の可視化。煩雑なフラグ管理の手助けになれば。
